# MediHome 法的コンプライアンスチェックリスト

**作成日**: 2025年12月15日

## 概要

MediHomeは情報提供サービスであり、不動産取引や医療行為の仲介・斡旋を行わないため、宅建業法・医療法の規制対象外です。ただし、誤解を招かないための表現ガイドラインと免責事項が必要です。

---

## 1. 宅地建物取引業法（宅建業法）

### 規制内容
- 不動産の売買・交換・賃貸の代理・媒介

### MediHomeの対応
- ✅ 不動産取引の仲介・斡旋を行わない
- ✅ 過去の取引価格データを情報提供のみ

### 必須対応

#### 免責事項（Phase 1で実装）
```markdown
## 免責事項

### 不動産情報について
- 本サービスで表示される不動産取引価格は、国土交通省の過去の取引事例データに基づく参考情報です
- 不動産取引の最終判断には使用しないでください
- 最新の価格や詳細情報は、不動産業者または不動産ポータルサイトでご確認ください
- 本サービスは不動産の売買・賃貸の仲介・斡旋を行いません
```

#### 表現ガイドライン
| 表現 | 判定 | 理由 |
|------|------|------|
| 「周辺の過去の取引価格は〇〇円です（参考）」 | ✅ OK | 客観的事実 |
| 「このエリアの相場感をつかめます」 | ✅ OK | 情報提供の範囲 |
| 「この物件がおすすめです」 | ❌ NG | 特定物件への誘引 |
| 「今が買い時です」 | ❌ NG | 取引の推奨 |

## 2. 医療法・医療広告ガイドライン

### 規制内容
- 医療機関による広告（虚偽・誇大広告の禁止）
- 特定医療機関への誘引

### MediHomeの対応
- ✅ 公的データ（厚労省医療情報ネット）の客観的情報提供
- ✅ 特定医療機関を推奨・誘引しない

### 必須対応

#### 免責事項（Phase 1で実装）
```markdown
### 医療情報について
- 本サービスで表示される医療施設情報は、厚生労働省「医療情報ネット」のオープンデータに基づく参考情報です
- 診療科目・診療時間などの最新情報は、各医療機関に直接ご確認ください
- 医療アクセススコアは、距離・施設数などの客観的指標に基づく参考値であり、医療の質や治療効果を保証するものではありません
- 本サービスは特定の医療機関を推奨・誘引するものではありません
```

#### 表現ガイドライン
| 表現 | 判定 | 理由 |
|------|------|------|
| 「半径1km以内に小児科が3件あります」 | ✅ OK | 客観的事実 |
| 「距離が近い順に表示します」 | ✅ OK | 客観的基準 |
| 「医療機関リスト（距離順）」 | ✅ OK | 客観的表示 |
| 「おすすめの病院」 | ❌ NG | 特定医療機関の推奨 |
| 「評価が高い医療機関」 | ❌ NG | 主観的評価 |
| 「医療機関ランキング」 | ❌ NG | 優劣の明示 |

## 3. 景品表示法（優良誤認・有利誤認の防止）

### 規制内容
- 実際よりも著しく優良・有利であると誤認させる表示の禁止

### MediHomeの対応
- スコア算出根拠の透明化
- 断定表現を避ける

### 必須対応

#### スコア表示の要件（Phase 1で実装）
```typescript
// スコアカードの表示例
{
  score: 75,
  label: "子育てスコア",
  description: "半径1km以内の小児科数・夜間診療対応などから算出",
  detailsLink: "/about/scoring",  // 算出根拠の詳細ページ
  disclaimer: "参考値です。最新情報は各医療機関にご確認ください。"
}
```

#### 表現ガイドライン
| 表現 | 判定 | 理由 |
|------|------|------|
| 「子育てスコア: 75点（算出根拠を見る）」 | ✅ OK | 根拠明示 |
| 「小児科が半径1km以内に3件あります」 | ✅ OK | 客観的事実 |
| 「データは2025年12月1日時点のものです」 | ✅ OK | 更新時期明示 |
| 「子育てに最適なエリアです」 | ❌ NG | 断定表現 |
| 「絶対に便利な立地」 | ❌ NG | 断定表現 |
| 「No.1エリア」 | ❌ NG | 根拠不明確な最上級表現 |

## 4. 個人情報保護法

### MediHomeの対応
- ✅ 公開されている施設情報のみを利用
- ✅ ユーザーの個人情報を収集・保存しない

### 推奨対応（Phase 2）
- プライバシーポリシーの作成
- Google Analytics等のCookie使用時の同意取得

---

## 5. 著作権法・オープンデータ利用規約

### 必須対応（Phase 1で実装）

#### 出典表示
```markdown
## データ出典

本サービスは以下のオープンデータを利用しています：

- **医療施設情報**: [医療情報ネット（厚生労働省）](https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/iryou/teikyouseido/index.html)
  - ライセンス: 政府標準利用規約（第2.0版）
  - データ更新: 2025年12月1日時点

- **不動産取引情報**: 不動産取引価格情報（国土交通省）をもとにMediHome作成
  - ライセンス: 政府標準利用規約（第2.0版）
  - データ更新: 四半期ごと
```

## まとめ

### 法的リスク評価

| 法律 | リスク | 対策 |
|------|--------|------|
| 宅建業法 | 低（情報提供のみ） | 免責事項明示、推奨表現を避ける |
| 医療法 | 低（公的データ利用） | 免責事項明示、ランキング避ける |
| 景品表示法 | 中（スコア表示） | 算出根拠の透明化、断定表現避ける |
| 個人情報保護法 | 低（個人情報なし） | プライバシーポリシー作成（推奨） |
| 著作権法 | 低（オープンデータ） | 出典表示、利用規約遵守 |

### 結論

**MediHomeは基本的に法律に抵触しません。** ただし、以下の対策が必須です：

1. ✅ 免責事項の明示（Phase 1）
2. ✅ 出典表示（Phase 1）
3. ✅ スコア算出根拠の透明化（Phase 1）
4. ✅ 適切な表現の使用（断定を避ける）
5. ⚠️ プライバシーポリシー作成（Phase 2推奨）
