  - サービス名に「不動産情報ライブラリ」を使用しない → ✅ OK（MediHome）
  - 国が作成したかのような表示をしない → ✅ OK

## 📊 実施したスクリプト

### 1. データ確認

```bash
npm run phase0:download
```

実データファイルの存在確認と件数表示。

### 2. データインポート

```bash
npm run phase0:import
```

73件の診療所データをSQLiteにインポート。施設情報と診療科情報を統合。

### 3. データ分析

```bash
npm run phase0:analyze
```

診療科分布、座標データ、施設タイプなどを分析。

### 一括実行

```bash
npm run phase0:all
```

上記3つを順次実行。
